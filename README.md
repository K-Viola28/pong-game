# Pong Game

A classic Pong game with AI opponent, realistic ball physics, and a retro neon aesthetic.

## How to Play

1. **Start the Game**: Click "Start Game" to begin
2. **Control Your Paddle**: Move the left paddle using:
   - **Mouse**: Move your cursor up/down
   - **Arrow Keys**: ↑ Up, ↓ Down
3. **Gameplay**: 
   - The right paddle is controlled by an intelligent AI opponent
   - Score points by bouncing the ball past your opponent
4. **Reset**: Click "Reset Score" to clear the scoreboard and start over

## Features

- 🎾 **Ball Physics**: Realistic movement with spin based on paddle hit location
- 🤖 **Intelligent AI**: Computer opponent that tracks the ball intelligently
- 💥 **Collision Detection**: Accurate collision detection for walls and paddles
- 📊 **Real-time Scoreboard**: Live score tracking
- ✨ **Retro Aesthetic**: Neon green gaming aesthetic with classic vibes

## Getting Started

### Installation

```bash
git clone https://github.com/K-Viola28/pong-game.git
cd pong-game
```

### Running the Game

Open `index.html` in your web browser to play.
